# Scoring

Hot News Radar uses a transparent heat score from 0 to 100.

| Factor | Max | Purpose |
| --- | ---: | --- |
| Freshness | 35 | Prioritize recent source publication times |
| Query relevance | 25 | Reward title, source, and summary matches |
| Source authority | 15 | Prefer primary, indexed, or stable source lanes |
| Cross-source consensus | 15 | Reward repeated independent sightings |
| Engagement | 10 | Reward visible community attention when available |

## Labels

| Score | Label |
| ---: | --- |
| 75+ | hot |
| 58-74 | rising |
| 42-57 | watch |
| below 42 | weak |

## Interpretation

The score ranks attention priority. It does not prove truth.

Use score with:

- source links,
- evidence class,
- source health,
- coverage gaps,
- publication time,
- manual review notes.

For high-stakes claims, primary-source verification is required even when the score is high.
